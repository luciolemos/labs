# Template Tech V4 Green

## Identificacao
- ID: tech-v4-green
- Linha: Tech V4
- Base tecnica: Slim 4 + Twig + Bootstrap 5

## O que este template e
Variacao visual verde da linha Tech V4 para landing pages e sites institucionais gerados pelo Labs.

## O que este template tem
- Estrutura Slim 4 pronta (public, routes, src, views, storage)
- Front-end completo com assets e animacoes
- Secoes de landing page prontas para personalizacao
- Script de conversao para WebP em scripts/convert-webp.php
- Metadados de catalogo em template.json (label, descricao e preview)

## Paleta e uso recomendado
- Direcao: verde tecnologica
- Indicado para: automacao, infraestrutura, sustentabilidade e operacoes

## Estrutura principal
- public/: front controller, .htaccess e assets
- routes/web.php: rotas Slim
- src/Controllers/HomeController.php: controller da home
- views/: templates Twig
- storage/: cache e logs
- template.json: metadados no painel

## Uso no painel Labs
1. Acesse /admin.
2. Selecione tech-v4-green na coluna de template.
3. Salve para criar ou reprovisionar.
